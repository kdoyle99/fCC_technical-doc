HTML Setup

Before you start writing out the contents of your webpage, you have to set up your HTML document with the proper elements. All elements need to be wrapped in < >, and most elements will require an opening tag and a closing tag. However, there are many elements that are self-closing, meaning it does not need a closing tag.

The first line of code should only include the self-closing element tag, !DOCTYPE html to declare the type of document, "HTML" being the type. Following this line, the next element required is html which will encapsulate all of the other elements. Typically, the first element to go inside the html element is a head element, which will usually include important information such as the title of the webpage, a description, and the character set.

HTML Body

Following the head, the body element defines all of the content that will be visible on the webpage. From here, everything you type inside elements should appear on the web document.

Here's a list of all of the elements that every webpage should include:

  • header
  • main
  • section
  • footer
